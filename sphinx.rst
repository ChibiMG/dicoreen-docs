Documentation avec Sphinx
=========================

Sphinx utilise le langage RST
Cheat Sheet : https://thomas-cokelaer.info/tutorials/sphinx/rest_syntax.html#code-block-directive

Installation
------------

Prérequis : avoir installé Python

.. code-block::

    pip install -U sphinx

Création du projet
------------------

.. code-block::

    sphinx-quickstart

Génération
----------

.. code-block::

    make html
